### Rischi Informatici

**Principali Minacce**

1. SQL Injection

- Manipolazione query database
- Accesso non autorizzato

2. Cross-Site Scripting (XSS)

- Inserimento script maligni
- Siti web vulnerabili

3. Phishing

- Furto credenziali
- Email/siti contraffatti

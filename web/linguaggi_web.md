# Linguaggi Web Principali

---

### 1. **HTML (HyperText Markup Language):**

- **Descrizione:** Linguaggio di markup utilizzato per strutturare il contenuto delle pagine web.
- **Ruolo:** Definisce elementi come titoli, paragrafi, collegamenti e immagini.

---

### 2. **CSS (Cascading Style Sheets):**

- **Descrizione:** Linguaggio per la presentazione e lo stile delle pagine web.
- **Ruolo:** Controlla l'aspetto visuale degli elementi HTML, inclusi layout, colori e font.

---

### 3. **JavaScript:**

- **Descrizione:** Linguaggio di scripting lato client.
- **Ruolo:** Aggiunge interattività alle pagine web, gestisce eventi utente, e comunica con il server.

---

### 4. **TypeScript:**

- **Descrizione:** Una versione tipizzata di JavaScript, compilata in JavaScript.
- **Ruolo:** Migliora la manutenibilità del codice attraverso l'aggiunta di tipi.

---

### 5. **PHP (Hypertext Preprocessor):**

- **Descrizione:** Linguaggio di scripting lato server.
- **Ruolo:** Esegue operazioni sul server, genera dinamicamente pagine web.

---

### 6. **Python:**

- **Descrizione:** Linguaggio di programmazione versatile.
- **Ruolo:** Utilizzato lato server per sviluppare applicazioni web, framework popolari includono Django e Flask.

---

### 7. **Ruby:**

- **Descrizione:** Linguaggio di programmazione interpretato e orientato agli oggetti.
- **Ruolo:** Utilizzato lato server, noto per il framework Ruby on Rails.

---

### 8. **Java:**

- **Descrizione:** Linguaggio di programmazione adatto a diverse piattaforme.
- **Ruolo:** Spesso utilizzato lato server in combinazione con framework come Spring.

---

### 9. **C#:**

- **Descrizione:** Linguaggio di programmazione sviluppato da Microsoft.
- **Ruolo:** Utilizzato con il framework ASP.NET per lo sviluppo web.

---

### 10. **Go (Golang):**

- **Descrizione:** Linguaggio di programmazione sviluppato da Google.
- **Ruolo:** Apprezzato per le prestazioni e la facilità di sviluppo web.

---

### 11. **Swift:**

- **Descrizione:** Linguaggio di programmazione sviluppato da Apple.
- **Ruolo:** Utilizzato per lo sviluppo di applicazioni web e mobili su piattaforme Apple.

---

### 12. **Kotlin:**

- **Descrizione:** Linguaggio di programmazione moderno che gira sulla JVM (Java Virtual Machine).
- **Ruolo:** Utilizzato per lo sviluppo web e mobile, specialmente su piattaforme Android.

---

### 13. **Rust:**

- **Descrizione:** Linguaggio di programmazione orientato alla sicurezza.
- **Ruolo:** Utilizzato per lo sviluppo web ad alte prestazioni.

Questi linguaggi sono utilizzati in combinazione per sviluppare applicazioni web complete, con HTML per la struttura, CSS per lo stile, e JavaScript (o altri linguaggi lato server) per la logica e l'interattività. 

La scelta del linguaggio dipende spesso dai requisiti del progetto, dalle preferenze del team di sviluppo e dalle prestazioni desiderate.
